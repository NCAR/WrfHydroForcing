# wrfhydroforcing 

### Anaconda Python Setup/Installation
---
1) Install anaconda python 3
2) Create a virtual env 
   $conda create -n wrf
3) Activate the virtual env 
   $conda activate wrf
4) Install esmpy
   $conda install -c conda-forge esmpy
5) Install the python module and its dependencies.
   $pip install .
